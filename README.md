## Instructions for Running the Server Locally

1. **Clone the Repository**:
   First, clone the repository to your local machine:

   ```bash
   git clone https://github.com/OctoDev8/simple-express-server.git
   cd simple-express-server

   ```

2. **Node**

Run node server.js
